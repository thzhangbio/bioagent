## 四类角色的 AI Agent 建设优先级

| 优先级 | 聚类 | 理由 |
|-------|------|------|
| **P0 最高** | D-新媒体/内容运营 | AI可替代性最高，内容批量生产场景明确，ROI最大 |
| **P1 高** | B-学术论文/SCI编辑 | 数据分析+文献检索+论文润色组合高度可AI化，市场需求大 |
| **P2 中** | C-医学推广/会务支持 | 物料初稿可AI化，但需要大量客户交互，适合人机协作模式 |
| **P3 低** | A-临床医学写作 | 专业壁垒最高，法规风险大，需要最深的领域知识和人工把关 |

### 建议的切入策略

**第一阶段：D类 Agent — 「AI 内容工厂」**
- 输入：热点话题/产品关键词/目标受众
- 输出：公众号推文、科普文章、短视频脚本
- 能力：选题策划 → 素材收集 → 初稿生成 → SEO优化 → 排版输出
- 估计覆盖率：80%+ 的D类岗位工作量

**第二阶段：B类 Agent — 「AI 学术助手」**
- 输入：研究方向/数据集/目标期刊
- 输出：文献综述、数据分析报告、SCI论文初稿、润色建议
- 能力：文献检索 → 数据分析 → 论文框架 → 初稿撰写 → 语言润色
- 估计覆盖率：60-70% 的B类岗位工作量

**第三阶段：C类 Agent — 「AI 医学推广助手」**
- 输入：产品Brief/会议信息/文献资料
- 输出：PPT框架、DA初稿、会议新闻稿、培训材料
- 能力：文献整理 → 要点提炼 → 物料初稿 → 格式适配
- 估计覆盖率：50-60% 的C类岗位工作量（剩余需人工客户沟通）

**第四阶段：A类 Agent — 「AI 临床写作辅助」**
- 输入：临床试验方案要素/法规模板/既往案例
- 输出：方案初稿、CRF模板、研究者手册框架
- 能力：模板填充 → 法规合规检查 → 初稿生成（需强力人工审核）
- 估计覆盖率：30-40% 的A类岗位工作量（法规风险高，需严格人审）
